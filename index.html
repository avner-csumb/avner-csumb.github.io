Welcome!

:sunny:

We are going to learn about several data structures, in addition to discussing runtime complexity (Big O) and problem-solving. We will be holding this bootcamp online. You will be provided with a number of resources and coding exercises. We will have a Slack chat and weekly Google Hangouts. (If you are not familiar with Slack, watch this video for the basics of sending messages.)

Prerequisites

Any code samples will use the Java programming language. To get the most out of this bootcamp, you will want to have a working knowledge of Java.

Evaluation

No college credit is offered for this bootcamp. It is offered to deepen your understanding and prepare you for internships and jobs. I will ask that you submit completion checklists, but these are on an honor code basis. You will not be submitting assignments and/or taking tests. For this bootcamp, your motivation will be learning and not grades or points.

Data Structures

Tenatively, we will ocver the following data structures:

  • Linked Lists (including doubly-linked)
  • Trees
  • Stacks
  • Queues
  • Heaps
  • Hash Tables

Java comes with many built-in data structures. For the purpose of this bootcamp, we will not use these, but rather will be building our own.
